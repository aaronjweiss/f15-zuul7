Project: Zuul-7
Authors: Michael Kölling, David J. Barnes, Aaron Weiss, Randolph Mitchell, Colin P. Goss

This project based on material for the book

   Objects First with Java - A Practical Introduction using BlueJ
   Fifth edition
   David J. Barnes and Michael Kölling
   Pearson Education, 2012
   
This project represents a basic adventure game. The player can move through the world and find items and NPCs along the way.

To start this application, create an instance of class "Game" and call its
"play" method.
